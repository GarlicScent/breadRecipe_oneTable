# 🥪Onetable

### 제빵 레시피 공유 사이트 제작

상세 소개: 코딩온 교육기관을 통해 진행한 세번째 프로젝트입니다. 요리 레시피 중 빵과 관련된 레시피를 공유하는 사이트를 제작하였으며, 2명씩 프론트와 백엔드로 역할을 나누어 4명이 한 조로 2주간 개발을 진행하였습니다.
